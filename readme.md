## 语雀搜索

用来检索自己的语雀知识库

## 使用
1. 选择性安装 [语雀减藏](https://github.com/yuquewebclipper/yuque-web-clipper)
2. 安装[语雀搜索](https://github.com/ylfeng250/yuque-search)
3. 设置自己的 token
4. 选择自己要检索的仓库
5. 在Chrome的地址栏输入`yuque`, 按下`tab`键就可以输入关键字搜索了
